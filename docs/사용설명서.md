# 깐돌이 전산등록 자동화 매크로 사용설명서

본 프로그램은 엑셀 파일의 고객 정보를 읽어 솔팅 프로그램 전산 등록 및 보험사 동의서 다운로드/서명 스탬핑, EDMS 업로드 작업을 자동화하는 매크로입니다.

---

## 1. 주요 기능

### 1단계: 솔팅 전산등록
* 엑셀에서 고객 리스트(주민번호, 이름, 전화번호)를 로드하여 솔팅프로그램에 자동 등록합니다.
* 등록 완료 및 성공 여부를 리포트에 실시간 기록합니다.

### 2단계: 보험사 동의서 출력 (KB손해보험)
* 보험사 사이트에 자동 로그인하여 고객 전산 등록을 수행합니다.
* 사이트 내 동의서 출력 창을 띄우고, OZ 리포트 뷰어를 제어하여 동의서 PDF 파일을 지정된 폴더에 자동으로 다운로드합니다.

### 3단계: 동의서 서명 스탬핑 (자동화)
* 다운로드된 동의서 PDF 파일에 미리 설정한 서명/스탬프 이미지를 올바른 위치에 자동으로 적용(합성)합니다.

### 4단계: EDMS 일괄 전송
* 서명 스탬핑이 완료된 PDF 파일들을 KB EDMS 스캔 시스템으로 자동 전송 및 일괄 등록합니다.

---

## 2. 사용 방법

1. **프로그램 구동**: `Kkandori.exe` 또는 `main.py`를 실행합니다.
2. **자격증명 설정**: `⚙️ 설정` 탭에서 포털 및 보험사 ID, 비밀번호, 생년월일을 입력한 후 **[설정 저장]** 버튼을 누릅니다.
   * *보안 안내: 입력한 비밀번호와 중요 정보는 암호화(Base64)되어 로컬 컴퓨터에만 안전하게 보관됩니다.*
3. **1단계 진행**: `1단계` 메뉴에서 **[포털 자동 로그인]**을 눌러 브라우저 디버그 세션을 시작하고 로그인을 완료합니다.
4. **2단계 진행**: `2단계` 메뉴에서 **[보험사 자동 로그인]**을 눌러 보험사 업무 시스템 세션을 연결합니다.
5. **3단계 진행**: `3단계` 메뉴에서 대상 엑셀 파일과 PDF 저장 경로를 로드한 뒤, **[▶️ 전산 처리 시작]**을 클릭합니다.
6. **작업 중단**: 작업 진행 중 중단이 필요할 경우, 프로그램 화면 우측 하단의 **[🛑 중단]** 버튼을 클릭하거나 단축키 **`Ctrl+D`**를 누르면 즉시 안전하게 루프가 종료됩니다.

---

## 3. 화면 구성 및 제어

* **실시간 로그 창**: 매크로가 실행되는 모든 세부 과정과 처리 성공/실패 여부를 마스킹 처리된 로그로 확인할 수 있습니다.
* **진행률 표시 바**: 전체 작업 대상 중 몇 번째 건이 수행 중인지 직관적인 그래프로 보여줍니다.
* **비율 조절 (Zoom) 옵션**: 노트북이나 저해상도 모니터에서도 화면이 깨지거나 가려지지 않도록 85% / 100% / 115% 화면 비율 조절을 실시간으로 지원합니다.
