# 깐돌이 자동화 매크로 사전세팅방법 가이드

매크로 프로그램이 원활하고 오류 없이 작동하기 위해서는 실행 전 컴퓨터 환경을 다음과 같이 사전에 설정해 주어야 합니다.

---

## 1. 시스템 요구사양 및 브라우저 설정

* **운영체제(OS)**: Windows 10 또는 Windows 11 (64bit 필수)
* **기본 브라우저**: **Microsoft Edge** 브라우저 설치 및 최신 버전 유지 권장
* **화면 배율**: 윈도우 디스플레이 설정에서 화면 텍스트 배율을 100%로 설정하는 것을 권장합니다. (해상도 1920x1080 이상 권장)

---

## 2. 필수 사전 설정 단계

### Step 1: Microsoft Edge 디버그 모드 구동
매크로 프로그램이 열려 있는 브라우저 창을 직접 제어하고 보안 로그인 세션을 유지하기 위해서는 Edge 브라우저가 원격 디버깅 모드로 켜져 있어야 합니다.
1. 바탕화면 또는 프로그램 폴더에 있는 **`start_edge_debug.bat`** 파일을 실행합니다.
2. 디버깅 모드로 실행된 Edge 브라우저 창이 뜨면 닫지 않고 켜둔 상태로 유지합니다.

### Step 2: 자격증명(ID/비밀번호) 입력 및 저장
1. 깐돌이 프로그램(`Kkandori.exe`)을 실행합니다.
2. 우측 상단의 `⚙️ 설정` 버튼 또는 설정 아이콘을 클릭합니다.
3. **포털 자동 로그인 설정** 및 **보험사 자동 로그인 설정** 영역에 각 사이트의 아이디, 비밀번호, 생년월일(6자리)을 정확히 기입합니다.
4. **[설정 저장]** 버튼을 눌러 입력값을 암호화 저장합니다. 이제 매번 실행 시 재입력할 필요 없이 자동으로 필드가 세팅됩니다.

### Step 3: 기본 프린터 및 동의서 저장 포맷 설정
OZ 리포트 뷰어에서 동의서를 PDF 가상 프린터로 저장하거나, 다중스캔 제한 우회를 위해 PNG 이미지 포맷으로 저장할 수 있습니다.
1. Windows 시작 메뉴에서 `프린터 및 스캐너`를 검색하여 엽니다.
2. 목록에 **`Microsoft Print to PDF`**가 있는지 확인하고 활성화해 줍니다.
3. 3단계 설정 화면에서 **동의서 형식**을 `Adobe PDF File (*.pdf)` 또는 `PNG Image File (*.png)`로 선택합니다.
   * 다중입력(10명씩 일괄 등록) 사용 시에는 PNG 이미지 형식 출력을 권장합니다.
4. **입력 방식**을 `단일입력 (1명씩)` 또는 `다중입력 (10명 일괄)` 중에서 선택해 구동합니다.

### Step 4: 서명 및 스탬프 이미지 배치
동의서 자동 스탬핑 기능을 사용하려면 서명(도장) 이미지 파일이 올바른 경로에 위치해야 합니다.
1. 준비된 도장 또는 서명 이미지(배경이 투명한 PNG 권장) 파일명을 `stamp.png`로 저장합니다.
2. 해당 파일을 프로그램 실행 경로 내 `solting_auto` 또는 `output/resources/` 폴더 아래 배치합니다.
3. 스탬프 적용 위치나 세부 조정이 필요할 경우 `config.yaml` 의 `stamping` 관련 옵션을 편집합니다.
